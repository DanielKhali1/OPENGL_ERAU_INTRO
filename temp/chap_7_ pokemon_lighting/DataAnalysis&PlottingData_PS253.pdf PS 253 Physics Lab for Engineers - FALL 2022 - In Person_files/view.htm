In this lab you will do basic exercises in data analysis and create several different types
of plots. After completing these exercises, you should have a better idea of what we
mean by a 'properly formatted plot' or a 'good plot'. Keep in mind that there are many
ways to display results to your readers and it would be impossible to cover them all
here, but this lab will cover the methods most used in this course.
Your graded work will consist of an in-lab worksheet and a post-lab worksheet.
Introduction

One of the primary objectives of this course is to give you experience in analyzing data and
communicating your work to others in a written format. These are two of the main
processes in the scientific method, and in technical fields the ability to visualize data
effectively and efficiently is particularly important. In some cases, you will use a plot to help
analyze your data. Plots primarily allow readers to quickly absorb the most important results
of your work. If you ask researchers how they read the published work of others, most will
tell you they read the abstract and then go straight to looking at the plots, figures, and data
tables before actually reading anything. This is where results are in their most readily
available form!
You should know that the word “plot” categorizes a very broad set of visual aids. Plots
generally need some number of well-defined and labeled axes, however this isn’t always the
case. For example, a common way to represent data in political science is to overlay statistics
on a map of the region.
The skills and habits learned in making scientific plots can be applied to other, less technical
pursuits as well. With the proliferation of relatively cheap devices capable of taking, storing,
and transmitting data, virtually every discipline or industry can now use vast amounts of data.
With all this data comes the need to access it in a straightforward way that is useable by
people and society. Combined with the continual evolution and improvement of the internet
and consumer electronics, having the skills to produce excellent visual data has never been
more important. The recent explosion of the use of modern infographics online by
everything from governments, companies, nonprofits, and individuals highlights the
continued need for expertise in data analysis and data visualization.

X-Y Scatter Plot Guidelines

1. Always make sure that the X and Y values have been assigned to the correct axis when
the software displays the plot! Otherwise your slope will, at minimum, be the inverse
of the value you are trying to find.
2. The X and Y axes are labeled with the variable name measured and its units.
3. The figure label number and title goes below the plot at the start of the caption. See
Figure 2 as an example.
4. The plot is briefly described with a caption. It must identify the system you tested and
possibly the test conditions. Someone should be able to look at the caption and know
as much as possible about the experiment. Do not use “Y vs. X” in the caption, Y and
X are already defined in the axis labels.
5. Choose your axis scale wisely so the entire plot area is used. See Figure 1.
6. We will use Excel spreadsheet software operations to plot points on plots, but we will
not use the point to point connection (connect-the-dots) feature of the spreadsheet for
experimental data.
a. Remember your slope value has units!!!
b. Never connect from dot to dot on a plot. The data are points, your analysis can tell
you if a curve or function might fit the data nicely.
c. Use the software to find the best fit model if appropriate; this is typically a linear
best fit line (trendline).

Visually analyze these while thinking on what was tested.

7. Click OK to close the Select Data Source window. Now investigate your plot and make

8. Add a Trendline to your plot by right clicking on any data point in your plot and then

9. Occasionally the default displayed trendline equation uses a poor choice of significant
digits. For example only showing “2x” as the slope or “1” as the y-intercept. You can
manually adjust this by right-clicking the trendline equation and selecting Format
Trendline Label. You can adjust decimal precision if you choose either Number or
Scientific under Label Options > Number > Category.
10. Your plot should now have data points and a best fit linear trendline along with a
displayed equation for the line. What is left to do is clean up the plot to make it more
presentable. With the plot selected, use the options under the Chart Tools top menu or
double click on the plot to bring up the Format Chart side menu.
You may add proper labels with units to the axes, delete the legend titled 'Series1' if
present, select and delete the horizontal/vertical gridlines in the plot area, double click
the axes or scale values to change them so the data spans the entire plot area. You may
also change the colors and line styles if you want.
11. It is also possible to add error bars to your data points. While having the graph selected,
in the Chart Tools menu area under Layout click on the icon that says Error Bars. You will
have several options to choose from; it is normally easiest to click one of the options

error bars on the data points. Once they are present you can double click on them or
back under Error Bars choose 'More Error Bars Options'.
12. Since we were given uncertainties for our measured variables the error bars would be

14. While in Excel, make sure you have any cell selected, not the plot. In the menu find
Data > Data Analysis > Regression and open it. All you need to do is select the input data
for x-values and the input data for y-values. I would also select what area of the

The output from Regression will give you a bunch of statistical information about a linear best
fit to your input X-Y data. You will be most interested in the Coefficients and Standard Error for
the cells labeled Intercept and X Variable. These should be the same values for the slope and y-
intercept of the equation in your plot, only to much higher precision. Added bonus: Excel has
calculated for you the propagated uncertainty on the slope and y-intercept!
Note the uncertainties found by Regression were not done by error propagation; instead it was
done using statistical analysis. The assumption is the input into the linear regression command
is in fact supposed to be linear. Excel then compares how far the data points vary from an
ideal linear fit. If the data points fall far from the fit in a random fashion then the calculated
uncertainty will be high. For small data sets, error propagation, like what we do by hand, might
give more reliable uncertainties than a statistical approach.
Although it does not take into account things like our original measurement uncertainties,

Note for Mac Users

For students using Excel 2013 or earlier2 on a Mac, unfortunately it does not come with the
option to activate the Analysis Toolpak so you cannot use this built in feature to perform
Regression or other statistical analysis of your data. There is a work around: you will need to
install a separate program called StatPlus which will perform the same function. The program
works by pulling data from an already open Excel file.
However, there is a weird quirk with the StatPlus input fields that you need to be aware of in
order to get correct results. When you select your columns to input x,y data into the regression
routine it DOES NOT use the data in the first selected rows of the columns you select. So
you must make your selection including one empty row at the top of your columns.
For example if you have two columns x=A & y=B and the data goes from rows 10-20 you
should select x values = A9-->A20 and y values = B9-->B20. Additionally if your data starts
in the first row you will probably want to copy/paste it down into at least the 2nd row so you
can do the selection and still include all the data. You can confirm it has used the correct
number of data points by looking at the "Residuals" area at the bottom of the output and
making sure the "observations" # matches the number of data points you have.
